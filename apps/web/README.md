# @griptix/web

This is the Next.js 14 frontend web application for the Griptix platform.

## Features
- **Global App Shell**: TopNav with debounced predictive search, Footer, and sliding CartDrawer.
- **Storefront**: SSR collection pages and Product Detail Pages (PDP) with SEO schemas.
- **Sizing Wizard**: 5-step anatomical sizing flow with direct Cloudflare R2 file uploads.
- **User Portals**: Customer dashboards and B2B partner account sections.

## Stack
- Next.js 14
- TypeScript
- TailwindCSS / Vanilla CSS
- Zustand (client-side state management)
- NextAuth.js (authentication)
- Stripe Elements & Razorpay Checkout Modal

-- PRELOADER SEQUENCE (Landing Page) --

PHASE 1 — INITIALIZATION
  - Preloader UI appears instantly on page load (no delay).
  - Full-screen dark overlay (#1A1613) with subtle radial gradient texture.
  - Centered inner container (#241F1A) with:
      • GRIPTIX wordmark (all-caps, clean sans-serif, metallic sheen sweep animation)
      • Tagline: "Precision in every Pull" (below wordmark, muted, letter-spaced)
      • Percentage counter text (e.g. "64%") — centered below tagline
      • Thin 1px horizontal progress bar — immediately below counter
  - Behind the overlay, the browser begins fetching: /public/3d/grip.glb

PHASE 2 — LOADING PROGRESS
  - Progress bar scales from 0% → 100% width (left to right, smooth).
  - Counter ticks upward (12%...45%...99%) in sync with bar.
  - GRIPTIX logo runs a continuous metallic sheen sweep (CSS gradient wipe).
  - GLB load is tracked via THREE.DefaultLoadingManager.
  - Fallback: if GLB is missing or takes > 4 seconds, auto-advance to Phase 3.

PHASE 3 — TRANSITION (THE REVEAL)
  - Counter reaches 100%. Brief 400ms pause so user registers completion.
  - Counter and progress bar fade to opacity 0.
  - Inner container slides upward off-screen (translateY -100vh, 0.8s cubic-bezier).
  - Overlay fades and is removed from the DOM.

PHASE 4 — BURST (3D EXPLOSION)
  - 10 cloned grip.glb instances burst outward from screen center.
  - Each starts at (0,0,0) and animates to a unique radial target position.
  - Timing: staggered per-instance delay (40ms apart) for organic feel.
  - Easing: easeOutCubic for natural deceleration.
  - Depth distribution across the Z-axis (z: 0 to -6):
      • Close (z=0):  large scale (0.82), fully opaque (opacity 1.0)
      • Far  (z=-6):  small scale (0.52), faded (opacity 0.35)

PHASE 5 — FLOAT & SETTLE
  - After burst, each grip enters a slow continuous float (sin wave, Y-axis).
  - Each grip slowly rotates on all three axes at unique speeds.
  - Float speed and phase are unique per instance (no synchronized bobbing).

PHASE 6 — UI REVEAL
  - As grips settle, the main GRIPTIX UI (TopNav + Hero + Stats) fades in.
  - Fade-in duration: 600ms ease-in.
  - The 3D canvas remains as the full-screen animated background.

-- LOGO DESIGN RULES --
- Always render as: GRIPTIX (all caps, no exceptions)
- Font: Clean sans-serif — no decorative flourishes, no serifs
- Character: Strong, authoritative, confident — all caps projects stability
- Do NOT use script, display, or decorative typefaces for the brand name
- Minimum size: never render smaller than 16px height to preserve legibility

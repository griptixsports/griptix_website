# @griptix/ui

This is the shared component library and design token repository for the Griptix platform.

## Features
- **Design Tokens (`tokens.ts`)**: Colors (accent Gold `#D4AF37`, Deep Black `#0A0A0A`), typography scale, spacing (4px scale), shadows, and border radii.
- **Component Kit**: 30+ stateless, prop-driven React components.
- **Storybook**: Local developer sandbox and static compilation target.
- **Dark/Light Mode**: Full CSS variables support for smooth theme toggles.

## Stack
- React
- TypeScript
- Storybook 7+
- TailwindCSS / Vanilla CSS
